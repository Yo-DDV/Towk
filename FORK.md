# Fork Chatto Responsive

Ce dépôt est le fork public `Yo-DDV/chatto-responsive` du projet `chattocorp/chatto`. Le fork conserve l’historique, les notices et les frontières de licence de l’amont. Il sert à construire et exploiter un pilote indépendant avant toute décision de renommage fonctionnel.

## Gouvernance

- `origin` : `https://github.com/Yo-DDV/chatto-responsive.git` ;
- `upstream` : `https://github.com/chattocorp/chatto.git` ;
- `main` reste protégée par des branches courtes et des pull requests ;
- les synchronisations amont conservent les commits originaux ;
- les adaptations propres au fork sont séparées des synchronisations ;
- aucune image du pilote ne porte le tag `latest`.

Le premier pilote utilise `towk.example`. Cette URL ne vaut pas validation définitive de la marque « Towk » : le plan de rebranding reste une décision distincte.
