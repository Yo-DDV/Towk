## AdminDiagnosticsService

Provides server diagnostics for the admin system page.

<a id="chatto-admin-v1-AdminDiagnosticsService-GetSystemInfo"></a>

### GetSystemInfo

Returns broker, JetStream, and projection diagnostics. Requires the
server owner role.

```http
POST /api/connect/chatto.admin.v1.AdminDiagnosticsService/GetSystemInfo
```

<a id="chatto-admin-v1-GetSystemInfoRequest"></a>

#### Input: GetSystemInfoRequest

Request to read owner-only server diagnostics.


<a id="chatto-admin-v1-GetSystemInfoResponse"></a>

#### Result: GetSystemInfoResponse

Owner-only operational diagnostics for the Towk server.

| Field | Type | Description |
| --- | --- | --- |
| `system_info` | [`AdminSystemInfo`](/reference/connectrpc-api/types/#chatto-admin-v1-AdminSystemInfo) | Broker connection, account, and JetStream state. |
