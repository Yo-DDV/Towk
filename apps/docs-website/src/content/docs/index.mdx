## What Towk provides

- **Rooms, direct messages, threads, reactions, search, and notifications** in
  one responsive web client.
- **File sharing and media handling**, with NATS storage by default and optional
  S3-compatible object storage.
- **Voice and video calls with screen sharing** through an operator-managed
  LiveKit deployment.
- **Roles and scoped permissions** for communities ranging from a few rooms to
  larger structured workspaces.
- **A compact self-hosted runtime** that can start with embedded NATS and grow to
  external NATS, multiple replicas, and dedicated infrastructure.
- **A documented privacy boundary**, including per-user encryption for message
  text and selected account fields plus crypto-shredding during account deletion.

Towk is pre-1.0 software. Pin the exact revision you deploy, keep tested
backups, and review changes before upgrading.
