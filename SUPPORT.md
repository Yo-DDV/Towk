# Support

Towk is pre-1.0 community software. The public repository does not currently
provide an SLA, guaranteed response time, or commercial support contract.

- Use GitHub Issues for reproducible Towk bugs and scoped feature proposals.
- Use the repository documentation and existing issues for self-hosting help.
- Use the private route in [SECURITY.md](SECURITY.md) for vulnerabilities.

Before opening an issue, remove secrets and personal data, search for duplicates,
and include the exact Towk commit or image digest, deployment model, expected
behavior, observed behavior, and minimal reproduction. General infrastructure
administration and unsupported third-party modifications may be redirected to
their respective projects.
